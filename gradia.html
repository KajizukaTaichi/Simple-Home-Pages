Gradia

Lisp like programming language that can give type annotation for gradual typing

Sample

Gradia 0.1.0
> (define '(x2 n) '(* n 2))
(lambda '(n) '(* n 2))
> (x2 4)
8
> (x2 " 5")
10
> (define '(x2 n:number) '(* n 2))
(lambda '(n:number) '(* n 2))
> (x2 6)
12
> (x2 "7 ")
Error! the passed argument value `"7 "` is different to expected type `number` of the function
>
